This is personal project about student placement result website. Data from kaggle (https://www.kaggle.com/datasets/sonalshinde123/student-placement-dataset/data).

website by F5

1. Setup environment Shell/Terminal <br>
   ```
   mkdiv proyek1
   cd proyek1
   cd proyek1
   pipenv install
   pipenv shell
   pip install pandas matplotlib
   pip install streamlit
   ```
2. Run steamlit app
   ```
   streamlit run navigasi.py
   ```
